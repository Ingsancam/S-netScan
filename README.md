# S-netScan
S-netScan is a basic network scanner developed in Python

S-netScan is a basic network scanner developed in Python as my final project for Stanford University's Code in Place. The program automatically detects the local IP address of the host device, scans the local network, identifies active devices, and allows the user to save the results in a report file.

I chose this project because I am interested in networking and cybersecurity, and I wanted to apply the programming concepts learned throughout the course to a practical tool. During development, I worked with functions, loops, conditionals, lists, strings, file handling, and Python modules such as socket and os.

One of the main challenges was obtaining the correct local IP address and using it to scan devices on the network. Solving this problem helped me better understand both Python programming and basic networking concepts.

This project allowed me to connect my academic interests in Communications and Electronics Engineering with the programming skills taught in Code in Place. Overall, it was a rewarding experience that strengthened my confidence in developing real-world applications with Python.
